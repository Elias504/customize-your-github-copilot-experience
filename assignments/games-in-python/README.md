
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, conditionals, random selection, and user input.

## 📝 Tasks

### 🛠️ Select a Mystery Word

#### Description
Create a predefined list of possible words and choose one random word at the start of the game.

#### Requirements
Completed program should:

- Store multiple candidate words in a Python list.
- Randomly select one word when the game begins.
- Keep the selected word hidden from the player.

### 🛠️ Handle Letter Guesses

#### Description
Prompt the player to guess one letter at a time and show the current progress of the hidden word.

#### Requirements
Completed program should:

- Accept a single letter from the player.
- Display the word progress using underscores for unguessed letters.
- Reveal correct guesses in every matching position.
- Prevent repeated guesses from being counted more than once.

### 🛠️ Track Wins and Losses

#### Description
Keep track of incorrect guesses and end the game when the player wins or runs out of attempts.

#### Requirements
Completed program should:

- Track how many incorrect guesses remain.
- End the game when the full word has been guessed or the attempts are exhausted.
- Display a win message when the player guesses the word.
- Display a lose message when the player runs out of attempts.
